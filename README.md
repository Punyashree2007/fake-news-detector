VERALINE is a simple rule-based fake news detection system built using Java.
It analyzes a news headline or text and classifies it as Real, Suspicious, or Fake using predefined logical rules.

This project does not use Machine Learning. Instead, it relies on keyword detection, tone analysis, punctuation patterns, and a scoring mechanism and no API's and Database is used.

Features

Detects suspicious keywords
Works offline
Lightweight and fast

Tech Stack

Java (OOP Concepts)
VS Code
Command Line Execution

How to Run
javac Server.java
java Server

Enter a news headline when prompted and get instant classification.

Purpose
This project demonstrates how a simple rule-based system can detect potential misinformation using core Java programming concepts.
